Magento Modules

Redis Manager

This module was inspired by a client with a very unique Redis caching setup. The Redis ring operated on several servers using master-slaves. Because each web host had a different Redis cache, normal cache clearing didn't always behave as expected.

With this module, you can configure any number of Redis endpoints, view statistics of each database, and selectively flush DBs.

Shell Enhancements

This is a single abstract class that I've been using for my Magento shell scripts. It's features include:

  • Colorized output
  • Progress bars
  • Execution timer
  • Automated logging

Calendar & Events

This module enables the admin to create multiple calendar grids or event lists. Each event can be customized and link to an event details page or a product listing. Some features include:

  • Manageable from the admin panel
  • Configurable color swatches
  • Recurring events
  • Website filter
  • Custom URL router

Valutec Giftcards

This module integrates Valutec's giftcard service with Magento. A giftcard prompt is available during checkout and applies funds as a discount to the order. Some features include:

  • Balance checker
  • Integrated as faux payment method
  • Use normal payment methods when funds are insufficient
  • Credit memo creation voids/refunds giftcard
  • Debug logging
  • Shell script for card management
