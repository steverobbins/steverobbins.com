Magento Modules

Redis Manager

php magento redis

This module was inspired by a client with a very unique Redis caching setup. The Redis ring operated on several servers using master-slaves. Because each web host had a different Redis cache, normal cache clearing didn't always behave as expected.

With this module, you can configure any number of Redis endpoints, view statistics of each database, and selectively flush DBs.

View on GitHub

Cross Sell

php magento

This module was built to emulate Amazon's cross sell design. Relationships between products can easily be configured, with an optional discount for buying them together.

This is different from bundle products in that no additional product records are created. Only two skus are added to the cart.

It supports simple and configurable products. When configurable options are available, a lightbox will appear prompting their selection.
